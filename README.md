# skill-product-router

A Claude skill that assesses a task and recommends the best Claude product for it.

## Trigger
Ask: `what should I use for this?`, `which product suits this?`, `Claude Code or chat?`, or any variant.

## What it does
- Scores the task against Claude.ai Chat, Claude Code, Cowork, and Claude in Chrome
- Outputs a structured pros/cons table with fit ratings
- Gives a clear recommendation with reasoning
- Includes approach tips specific to the task and recommended product

## Files
- `SKILL.md` — skill instructions
